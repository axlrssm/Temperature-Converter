# 🌡️ Temperature Converter Web App

A simple and responsive **Temperature Converter** built using **HTML, CSS, and JavaScript**.  
This project allows users to convert temperature values between **Celsius (°C), Fahrenheit (°F), and Kelvin (K)** instantly.

---

## 🚀 Features

- 🔢 Input temperature values easily
- 🔄 Convert between:
  - Celsius (°C)
  - Fahrenheit (°F)
  - Kelvin (K)
- 📤 Select both:
  - Initial unit
  - Target unit
- ⚡ Instant conversion output
- 📱 Simple and responsive UI design
- 💻 Built with pure frontend technologies (no libraries/frameworks)

---

## 🧠 How It Works

1. User enters a temperature value
2. User selects the **original unit**
3. User selects the **target unit**
4. JavaScript calculates the converted value using standard temperature conversion formulas
5. Result is displayed instantly on the page

---

## 🛠️ Built With

- HTML5 – structure
- CSS3 – styling and layout
- JavaScript (Vanilla JS) – conversion logic
